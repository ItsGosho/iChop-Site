# Front-End

{% hint style="info" %}
The project is tested on **npm** version 6.12.0
{% endhint %}

```
npm install
```

{% hint style="info" %}
If you want to changed the port ,navigate to **.env** ,under the root folder
{% endhint %}

After the modules are installed ,the last thing is to run the project

```text
npm start
```
